# Рекомендации по развитию

## Стандарты кодирования

### Форматирование C# кода

- **Отступы:** 3 пробела (не табуляции)
- **Стиль скобок:** Allman style
- **Using statements:** отсортированы в порядке:
  1. System
  2. Microsoft.*
  3. Собственные namespaces
- **Пробелы после ключевых слов:** `if (`, `while (`, `for (`
- **Именование:**
  - Классы и методы: `PascalCase`
  - Приватные поля: `_camelCase`
  - Локальные переменные: `camelCase`

### Форматирование XAML

- **Отступы:** 3 пробела
- **Комментарии:** `<!--  текст  -->` (с двойными пробелами)
- **Многострочные атрибуты:** переносить на новую строку для читаемости
- **Структура ресурсов:** группировать по типам (цвета, кисти, стили)

### Комментарии

- **C# код:** одиночные строки `// Комментарий`
- **Многострочные C#:** `/* Комментарий */`
- **XAML код:** `<!--  Комментарий  -->`

## Архитектурные стандарты

### MVVM паттерн

- ViewModel должна содержать всю бизнес-логику
- View должна содержать только UI логику
- Использовать DataContext для привязки ViewModel к View

### Dependency Injection

- Регистрировать все сервисы в `App.xaml.cs` в методе `ConfigureServices`
- Использовать конструкторы для инъекции зависимостей
- Предпочитать `IInterface` вместо конкретных типов

### Асинхронное программирование

- Все IO операции должны быть асинхронными
- Методы async должны заканчиваться на `Async`
- Использовать `await` вместо `.Result`

## Версионирование и коммиты

- **Git flow:** develop → release → main
- **Коммиты:** писать на английском языке в повелительном наклонении
  - ✅ `Add user authentication feature`
  - ❌ `Added user authentication`
  
- **Теги:** использовать semantic versioning (v1.0.0)

## Требования к Pull Requests

- Описывать что было изменено и почему
- Следовать стандартам кодирования
- Добавлять unit тесты для новой функциональности
- Проходить code review перед мержом

## Инструменты

- **IDE:** Visual Studio 2026 или выше
- **Formatter:** использовать встроенный форматер VS (__Format Document__ или Ctrl+K, Ctrl+D)
- **EditorConfig:** автоматически применяется при открытии файлов
